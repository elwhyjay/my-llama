## 개발 가이드

### Git Commit 메시지 템플릿 사용하기

이 프로젝트는 일관된 커밋 메시지 형식과 TODO 추적을 위해 `.gitmessage` 템플릿을 사용합니다.

템플릿을 활성화하려면 다음 명령을 실행하세요:

```bash
git config --local commit.template .gitmessage
```

이후 `git commit` 명령을 실행하면 템플릿이 자동으로 로드됩니다.
